    b. How does absolute positioning depend on its parent element?

    c. How do you differentiate sticky from fixed (you can research on sticky)?

    d. If you were designing a webpage for a school event, how might you use positioning to highlight important information? Please give concrete examples.

    
    ANSWERS:

    a. 
    Static
    - Elements follow the flow of the document/website. 
    Relative
    - Elements maintains the following of the flow, but can be put back to it's original position using top, left etc...
    Absolute
    - Element breaks the normal flow and instead follows it's new element parent. Just like the notice CSS element where it sticks to the Main Content.
    Fixed
    - Element stays where it is positioned on the screen. Even if you scroll, it will remain in the same place.

    b. Absolute positioning depend on its parent element by not being able to exit or only having the ability to stick to it's sides/edges at max.

    c. Fixed and Sticky are different because Sticky is a hybrid between fixed and relative. It kind of just "sticks" in place when you scroll down. Unlike fixed where it just stays there on your screen.

    d. I would use CSS positioning maybe like fixed to really highlight important and critical information then maybe sticky as an amaze and an engagement feature of the viewers of my website.
